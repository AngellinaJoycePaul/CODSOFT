# Personal Portfolio Website

This repository contains my personal developer portfolio website built using HTML and CSS.  
It showcases my background, education, certifications, technical skills, and featured projects.

## Features

- Clean and responsive portfolio design
- Intro loading screen animation
- About Me section
- Education background
- Skills overview
- Certifications showcase
- Featured Projects with links
- Contact information and professional profiles
- Downloadable resume

## Technologies Used

- HTML5
- CSS3
- Google Fonts
- GitHub Pages (for deployment)

## Featured Projects

### Dual Mode Personal Safety Tracker
An IoT safety device developed using Arduino, GPS, GSM module, and accelerometer to send emergency alerts with location.

### Automated Currency Converter
A C-based financial simulation system capable of converting currencies with transaction fees, rate fluctuation, and transaction history.

### Landing Page
A responsive landing page built using HTML and CSS focusing on clean UI design and responsive layout.
